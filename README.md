# 📝 Estrelas do Amanhã — Formulário de Matrícula

> Formulário completo para matrícula escolar, desenvolvido para praticar a estruturação de dados complexos em HTML e o uso de recursos modernos de CSS como Grid e Nesting.

-----

## 📋 Sobre o Projeto

O **Estrelas do Amanhã** é uma aplicação web de formulário de matrícula para uma escola de educação infantil. O projeto foca na organização de informações em etapas (dados da criança, do responsável, opções de turno e documentos), utilizando uma interface limpa e intuitiva. Foi desenvolvido como parte da formação da **Rocketseat** para consolidar conhecimentos em **CSS Grid**, **Flexbox** e **Nesting nativo do CSS**.

Visualize \<a href="[https://wallissondev.github.io/formulario-de-matricula-rocketseat/](https://www.google.com/search?q=https://wallissondev.github.io/formulario-de-matricula-rocketseat/)"\>clicando aqui\!\</a\>

-----

## ✨ Funcionalidades

  - ✅ Organização por seções utilizando `<fieldset>` e `<legend>`
  - ✅ Campos para dados pessoais, endereço e informações de contato
  - ✅ Seleção de turno de estudo (Matutino/Vespertino) via rádio buttons customizados
  - ✅ Upload de documentos (Certidão de Nascimento) com área de drop personalizada
  - ✅ Validação de campos obrigatórios com feedback visual nativo
  - ✅ Layout responsivo adaptado para diferentes tamanhos de tela
  - ✅ Uso de ícones SVG integrados para melhor experiência do usuário
  - ✅ Tipografia moderna utilizando fontes do Google Fonts

## 🧠 Foco Técnico — Estruturação e Estilo

Este projeto serviu como um estudo aprofundado sobre como lidar com formulários que exigem muitos dados. Alguns conceitos aplicados:

  - **Semântica avançada:** Uso rigoroso de labels, IDs e agrupamentos para acessibilidade.
  - **Layout com Grid:** Posicionamento preciso de campos de diferentes tamanhos (ex: cidade, estado, CEP na mesma linha).
  - **CSS Nesting:** Organização do código CSS de forma mais limpa e hierárquica diretamente no arquivo `.css`.
  - **Customização de Componentes:** Estilização de inputs de arquivo e rádio para fugir do padrão nativo dos navegadores.
